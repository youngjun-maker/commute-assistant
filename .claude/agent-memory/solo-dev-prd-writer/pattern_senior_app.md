시니어 타겟 앱 PRD 작성 시 반복되는 패턴:

**UI/UX 필수 기준:**
- 최소 폰트 크기: 본문 18px, 핵심 정보 32px 이상
- 터치 타겟: 48x48px 이상 (Apple HIG 기준)
- WCAG AA 명암비 4.5:1 이상
- 화면당 최대 3개 주요 정보, 스크롤 최소화
- 약어/영어 축약어 금지, 풀어쓰기 필수

**MVP 범위에서 제외하는 것들:**
- 자가용/택시 경로 (대중교통 사용자 타겟 시)
- 소셜 로그인 (Supabase 매직링크로 충분)
- 실시간 GPS 위치 추적 (배터리 + 프라이버시 문제)
- 날씨 정보 연동 (핵심 흐름과 무관)

**중요 설계 포인트:**
- GPS 대신 등록된 고정 주소(집)를 출발지로 사용 → 위치 권한 불필요
- 시간대 기반 모드 자동 전환 (오전=출근 모드, 오후=퇴근 모드)
- 요일별 고정 스케줄 + 날짜별 오버라이드 테이블 분리 설계
